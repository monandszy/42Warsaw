# This project has been created as part of the 42 curriculum by sandrzej

## Description
The Netpractice project is a 10 level networking challenge. 10 exported configuration files (one per level) must be placed at the repository root.

## Instructions
Download the newest version of the netpractice tool. Run the levels by opening in the browser the index.html, or a given level.html file.
